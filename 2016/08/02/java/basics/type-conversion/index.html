java基础知识——数据类型转换

发表于 2016-08-02   |   分类于 java

在 Java 程序中，不同的基本数据类型的数据之间经常需要进行相互转换。

自动类型转换

代码中 int 型变量 score1 可以直接为 double 型变量 score2 完成赋值操作，运行结果为： 82.0

这种称为自动类型转换。

自动类型转换的条件

  1. 目标类型与源类型兼容，如 double 型兼容 int 型，但是 char 型不能兼容 int 型。
  2. 目标类型大于源类型，如 double 类型长度为8字节，int 类型为4字节，因此 double 类型的变量可以直接存放 int 类型的数据，但反过来就不可以了。

强制类型转换

尽管自动类型转换是很方便的，但并不能满足所有的编程需要。例如，当程序中需要将 double 型变量的值赋给一个 int 型变量，这种转换是不会自动进行的，因为 int 型的存储范围比 double 型的小。此时就需要通过强制类型转换来实现了。

强制类型转换语法

语法：(数据类型)数值

运行结果：
75.8
75

注意：通过强制类型转换将75.8赋值给int型变量后，结果为75，数值上并未进行四舍五入，而是直接将小数位截断。因此，强制类型转换可能会造成数据的丢失，使用时应慎重。
