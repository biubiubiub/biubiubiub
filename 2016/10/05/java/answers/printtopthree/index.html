输出前三名的成绩

发表于 2016-10-05   |   分类于 题解

题目

Description

输入指定个数的成绩，排序并输出前三名的成绩。并且可以判断无效的成绩。

Input

成绩的个数和所有的成绩

Output

前三名的成绩

Sample Input

5
99 53 71 66 80

Sample Output

第1名的成绩是：99
第2名的成绩是：80
第3名的成绩是：71

题解

可以倒叙遍历数组，将数组从大到小进行排列 for(int i=score.length-1;i>=0;i–)
