逆序的三位数

发表于 2016-07-14   |   分类于 题解

题目

Description

程序每次读入一个正三位数，然后输出逆序的数字。注意，当输入的数字含有结尾的0时，输出不应带有前导的0。比如输入700，输出应该是7。

Input

你的程序每次读到一个3位的正整数。

Output

输出逆序的数。

Sample Input

163

Sample Output

361

题解

a、b、c分别是原数的的百位、十位、个位，r、s、t分别是逆序的数的百位、十位、个位,x、y分别是原数和逆序数。

代码

```c
#include <stdio.h>
int main()
{
    int x,y,a,b,c,r,s,t;
scanf("%d",&x);
a=x/100;
b=(x-a*100)/10;
c=x-a*100-b*10;
r=c;
s=b;
t=a;
y=r*100+s*10+t;
printf("%d\n",y);
return 0;
}
```c
