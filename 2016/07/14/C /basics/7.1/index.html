C语言基础知识——循环

循环

while循环

  • 如果我们把while翻译作“当”,那么一个while循环的意思就是:当条件满足时,不断地重复循环体内的语句。
  • 循环执行之前判断是否继续循环，所以有可能循环一次也没有被执行。
  • 条件成立是循环继续的条件。
  • 循环体内要有改变条件的机会。
while(循环条件)
{
    循环体语句
}

do-while循环

  • 在进入循环的时候不做检查，而是在执行完一轮循环体的代码之后，再来检查循环的条件是否满足，如果满足则继续下一轮的循环，不满足则结束循环。
do
{
    循环体语句
}
while(循环条件)；

两种循环的关系

  • 相同点：都是条件满足时执行循环，条件不满足时结束循环。
  • 不同点：do-while在循环体执行结束时才判断条件，循环会至少执行一遍。while先判断再执行循环体，循环可能不会执行。
