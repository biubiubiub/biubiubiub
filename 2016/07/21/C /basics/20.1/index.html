C语言基础知识——函数

函数的定义

什么是函数？

  • 函数是一块代码,接收零个或多个参数,做一件事情,并返回零个或一个值
  • 可以先想象成数学中的函数 y=f(x)

为什么要用函数？

  • 可以重复使用，减少繁冗的代码，有需要时方便修改。

函数定义


函数的使用

调用函数

  • 函数名(参数值)
  • ()的作用是 表示函数调用
  • 即使没有参数也要用()
  • 如果有参数，则需要给出正确的数量和顺序，这些值会被按照顺序依次来初始化函数中的参数

函数返回

  • 函数知道每一次是哪里调用它,会返回到正确的地方

从函数中返回值
  • return停止函数的执行，并返回一个值
  • 可以没有返回值 return;
  • return 表达式;
  • 一个函数里可以出现多个return语句(不推荐)
  • 返回值可以再赋值给变量，可以再传递给函数，甚至可以丢弃

没有返回值的函数

  • void 函数名(参数表)
  • 不能使用带值的return
  • 可以没有return
  • 调用的时候不能做返回值的赋值
  • 如果函数有返回值，则必须使用带值的return


函数的参数

函数的先后关系
  • 像这样把sum()写在上面，是因为：
    • c的编译器自上而下顺序分析代码
    • 在看到sum(1,10)的时候，编译器需要知道sum()要几个参数，每个参数的类型，返回什么值
    • 从而检查对sum()的调用是否正确
  • 如果把sum()写在下面
    • 旧标准会假设所调用的参数都是int，返回也是int

函数原型

参数传递

调用函数

  • 如果函数有参数,调用函数时必须传递给它数量、类型正确的值
  • 可以传递给函数的值是表达式的结果,这包括:字面量、变量、函数的返回值、计算的结果

类型不匹配？

  • 调用函数时给的值与参数的类型不匹配是C语言传统上最大的漏洞
  • 编译器总是悄悄替你把类型转换好,但是这很可能不是你所期望的
  • 后续的语言,C++/Java在这方面很严格

参数传递过去的是什么？

  • C语言在调用函数时，永远只能传值给函数

传值

  • 每个函数有自己的变量空间,参数也位于这个独立的空间中,和其他函数没有关系
  • 过去,对于函数参数表中的参数,叫做“形式参数”,调用函数时给的值,叫做“实际参数”
  • 由于容易让人误会实际参数就是实际在函数中进行计算的参数,误会调用函数的时候把变量而不是值传进去了,所以不建议继续这种古老的方式来称呼它们
  • 它们是参数和值的关系


函数的变量

本地变量

  • 函数的每次运行,就产生了一个独立的变量空间,在这个空间中的变量,是函数的这次运行所独有的,称作本地变量
  • 定义在函数内部的变量就是本地变量
  • 参数也是本地变量

变量的生存期和作用域

  • 生存期:什么时候这个变量开始出现了,到什么时候它消亡了
  • 作用域:在(代码的)什么范围内可以访问这个变量(这个变量可以起作用)
  • 对于本地变量,这两个问题的答案是统一的:大括号内，也叫做“块”

本地变量的规则

  • 本地变量是定义在块内的
    • 它可以是定义在函数的块内
    • 也可以是定义在语句的块内
    • 甚至可以随便在一对大括号内来定义变量
  • 程序运行进入这个块之前,其中的变量不存在,离开这个块,其中的变量就消失了
  • 块外面定义的变量在里面仍然有效
  • 块里面定义了和外面同名的变量则掩盖了外面的
  • 不能在一个块内定义同名的变量
  • 本地变量不会被默认初始化
  • 参数在进入函数的时候被初始化了


其他细节

没有参数时

  • void f(void)是明确没有参数
  • void f()在传统C中，它表示f函数的参数表未知，并不表示没有参数

调用函数时的逗号和逗号运算符怎么区分?

  • 调用函数时的圆括号里的逗号是标点符号,不是运算符
  • f(a,b)这里面的逗号是标点符号
  • f((a,b))这里面的逗号是逗号运算符

函数里的函数？

  • C语言不允许函数嵌套定义

关于main

  • int main()也是一个函数
