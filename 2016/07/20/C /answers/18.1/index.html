正序分解整数

题目

Description

输一个非负整数,正序输出它的每一位数字

Input

一个非负整数

Output

按照正序排列的每一位数字

Sample Input

13425

Sample Output

1 3 4 2 5

题解

13425/10000=1————1
13425%10000=3425
10000/10=1000
3425/1000=3————–3
3425%1000=425
1000/10=100
425/100=4—————-4
425%100=25
100/10=10
25/10=2——————2
25%10=5
10/10=1
5/10=5——————-5
5%10=5
1/10=0
5/0=0
