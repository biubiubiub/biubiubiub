求最大公约数

题目

Description

输入两个正整数a和b，求出它们的最大公约数

Input

两个正整数

Output

它们的最大公约数

Sample Input

12 18

Sample Output

6

题解

枚举法

  1. 设t=1
  2. 若a和b都能被t整除,则记下这个t
  3. t=t+1,重复第二步直到t等于a和b中较小的那个
  4. 那么，曾经记下的最大的能整出a和b的t就是所求的最大公约数

辗转相除法

  1. 如果b=0，计算结束，a就是最大公约数
  2. 否则，计算a除以b的余数，再让a等于b，b等于那个余数
  3. 回到第一步
