数的逆序

Description

程序每次读入一个正整数，然后输出逆序的数字。注意，当输入的数字含有结尾的0时，输出不应带有前导的0。比如输入700，输出应该是7。

Input

你的程序每次读到一个正整数。

Output

输出逆序的数。

Sample Input

12345

Sample Output

54321

题解

用到了整数的分解和循环的知识。
