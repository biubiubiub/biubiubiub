健身最为有效的时间

下午4点到6点左右进行健身最为合适

训练组数

健身组数的决定，是来自训练的目的。

以肌肉体积增长为目的

如果健身的目的是肌肉体积增长，也就是长些肌肉塑形。
那么他的公式是（训练重量训练次数训练组数）
通过一系列科学的推导，最后发现，最有利于肌肉增长的安排是：训练重量（80%max）训练次数（8-12个）训练动作组数（4-8组）
由于80%12RM已经是动作的极限，所以我们尽量在训练重量中采取80%极限的负重。
这也就是为什么大多数人做的都是10个一组，每个动作3组。每次4-8个动作。这就是现代健美的韦德体系。

以力量增长为目的

如果目的是力量增长，决定的因素是训练重量。
比如举重选手或力量举选手，他们就常常采用80-100%max的重量做1-5个来训练。（一开始做热身，后面会中小重量练习技术细节。）
也就是力量增长的最佳安排一般是80-100%max1-5RM.有时只做1-3个动作。

组间休息

一般来讲，一组12RM的动作在60s左右（视动作而言），组间休息30-90s是最佳肌肉体积增长间隔

工具sizeof

是一个运算符，给出某个类型或变量在内存中所占的字节数
sizeof(int)算出类型的字节
sizeof(i)算出变量的字节
是静态运算符，它的结果在编译时刻就决定了
不要在sizeof的括号里做运算，这些运算是不会运行的。如sizeof(a++),括号里a++是不会运行的

考试的基本情况

去考试的准备

  • 周六上午8点50分以后开考。偶尔在周四，建议在周六，因为周六正常一些，周四可能会出现小概率事件，如考官进行实验。
  • 7点30到考场，尤其是第一次考试，准备好东西后进入考场坐等，一般来讲8点半左右考官会开始让考生填写个人信息。
  • 要保证8点50分是清醒的，每天大概要保证7点起床。

考试时间

  • 40分钟。
  • 考试时间分为两部分，30分钟听录音写答案，10分钟誊写答题纸。

考题

  • 40个题分为 section1 section2 section3 section4

题型

  • 填空题：最基本题型。不会少于15道，常见是20个左右。基本上是检验考生能不能在听完发音后写对单词。
  • 单选题：听懂了再选，不是听到某个单词跟选项相符就选。

输出考试成绩的前三名

发表于 2016-08-06   |   分类于 题解

题目

Description

编写一个 JAVA 程序，实现输出考试成绩的前三名
要求：
1、 考试成绩已保存在数组 scores 中，数组元素依次为 89 , -23 , 64 , 91 , 119 , 52 , 73
2、 要求通过自定义方法来实现成绩排名并输出操作，将成绩数组作为参数传入
3、 要求判断成绩的有效性（ 0—100 ），如果成绩无效，则忽略此成绩

Input

所有考试成绩。

Output

考试成绩的前三名。

Sample Input

89 ， －23 ， 64 ， 91 ， 119 ， 52 ， 73

Sample Output

考试成绩的前三名为：
91
89
73

题解

一、定义一个包含整型数组参数的方法，用来接收成绩数组，进行成绩排序并输出前三名。

1.在方法中使用Arrays类的sort()方法对数组进行排序，默认按升序排列，注意Arrays类的使用需要导入java.util.Arrays
2.由于只需要输出考试成绩的前三名，所以定义一个变量，统计有效成绩的前三名的数量
3.使用for循环遍历数组中的元素，由于要输出的是前三名成绩，所以从后往前遍历，即倒叙遍历。如 for( int i = scores.length - 1 ; i >= 0 ; i– ){}
4.判断成绩的有效性，如果有效成绩小于0或大于100，则使用continue忽略此成绩
5.如果成绩有效，则有效成绩加1。判断有效成绩的个数，如果有效成绩数大于3，则结束循环，只输出成绩的前三名

二、完成main方法

1.定义一个成绩数组scores，保存预定的考试成绩89，-23，64，91，119，52，73
2.调用自定义方法，传入成绩数组，完成功能

什么是方法？

其中

1.访问修饰符：方法允许被访问的权限范围，可以是public、protected、private甚至可以省略，其中public表示该方法可以被其他任何代码调用
