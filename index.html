求平均数（数组）

发表于 2016-07-22   |   分类于 题解

题目

Description

计算用户输入的数字的平均数,并输出所有大于平均数的数。（输入-1时程序停止运行）

Input

多个正整数（不确定个数）

Output

这些正整数的平均数和大于平均数的数

Sample Input

1 2 3 －1

Sample Output

平均数是2.000000 大于平均数的有：3

题解

必须先记录每一个输入的数字,计算平均数之后,再检查记录下来的每一个数字,与平均数比较,决定是否输出
